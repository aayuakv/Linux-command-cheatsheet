Basic Linux Commands
  1. pwd Command: This is used to display the location of the current working directory. 
    pwd
  2. mkdir Command: This is used to create a new directory under any directory. 
    mkdir <directory name>
  3. rmdir Command: This is used to delete a directory. 
    rmdir <directory name>
  4. ls Command: This is used to display a list of content of a directory. 
    ls
  5. ls Command: This is used to display a list of content of a directory with secret files. 
    ls -a
  6. cd Command: This is used to change the current directory. 
    cd <directory path>
  7. cd Command: This is used to go back to the previous directory. 
    cd ..
  8. touch Command: This is used to create empty files. We can create multiple empty files by executing it once. 
    touch <file name>
  9. cat Command: This is a multi-purpose utility in the Linux system. It can be used to create a file, display content of the file, copy the content of one file to another file, and more. 
    cat <file name>
            // Enter file content
  10. rm Command: This is used to remove a file. 
    rm <file name>
  11. cp Command: This is used to copy a file or directory. 
    cp <existing file name> <new file name>
  12. mv Command: This is used to move a file or a directory form one location to another location. 
    mv <file name> <directory path>
  13. rename Command: This is used to rename files. It is useful for renaming a large group of files. 
    rename 's/old-name/new-name/' files
  14. head Command: This is used to display the content of a file. It displays the first 10 lines of a file. 
    head <file name>
  15. tail Command: This is similar to the head command. The difference between both commands is that it displays the last ten lines of the file content. It is useful for reading the error message. 
    tail <file name>
  16. tac Command: This is the reverse of cat command, as its name specified. It displays the file content in reverse order (from the last line). 
    tac <file name>
  17. more command: The more command is quite similar to the cat command, as it is used to display the file content in the same way that the cat command does. 
    more <file name>
  18. less Command: is similar to the more command. It also includes some extra features such as 'adjustment in width and height of the terminal.' 
    less <file name>
  19. su Command: The su command provides administrative access to another user. In other words, it allows access of the Linux shell to another user. 
    su <user name>
  20. id Command: This is used to display the user ID (UID) and group ID (GID). 
    id
  21. useradd Command: This is used to add or remove a user on a Linux server. 
    sudo useradd <username>
  22. passwd Command: This is used to create and change the password for a user. 
    sudo passwd <username>
  23. groupadd Command: This is used to create a user group. 
    sudo groupadd <group name>
  24. cut Command: The cut command is used to select a specific column of a file. The '-d' option is used as a delimiter, and it can be a space (' '), a slash (/), a hyphen (-), or anything else. And, the '-f' option is used to specify a column number. 
    cut -d(delimiter) -f(columnNumber) <fileName>
  25. comm command: The 'comm' command is used to compare two files or streams. By default, it displays three columns, first displays non-matching items of the first file, second indicates the non-matching item of the second file, and the third column displays the matching items of both files. 
    comm <file1> <file2>
  26. wc command : The wc command is used to count the lines, words, and characters in a file. 
    wc <file name>
  27. od -b Command: This is used to display the content of a file in Octal format . 
    od -b <fileName>
  28. od -t x1 Command: This is used to display the content of a file in Hexa decimal format. 
    od -t x1 <fileName>
  29. od -c Command: This is used to display the content of a file in ASCII character format. 
    od -c <fileName>
  30. sort Command: This is used to sort files in alphabetical order. 
    sort <file name>
  31. gzip Command: This is used to truncate the file size. It is a compressing tool. It replaces the original file by the compressed file having '.gz' extension. 
    gzip <file1> <file2> <file3>
  32. gunzip Command: This is used to decompress a file. It is a reverse operation of gzip command. 
    gunzip <file1> <file2> <file3>
  33. find Command: This is used to find a particular file within a directory. It also supports various options to find a file such as byname, by type, by date, and more. 
    find . -name "*.pdf"
  34. locate command: This is used to search a file by file name. It searches the file in the database, whereas the find command searches in the file system. 
    locate <file name>
  35. date Command: This is used to display date, time, time zone, and more. 
    date
  36. cal Command: This is used to display the current month's calendar with the current date highlighted. 
    cal
  37. sleep Command: This is used to hold the terminal by the specified amount of time. By default, it takes time in seconds. 
    sleep <time>
  38. time Command: This is used to display the time to execute a command. 
    time
  39. zcat command: This is used to display the compressed files. 
    zcat <file name>
  40. df command This is used to display the disk space used in the file system. It displays the output as in the number of used blocks, available blocks, and the mounted directory. 
    df
  41. mount command: This is used to connect an external device file system to the system's file system.. 
    mount -t type <device> <directory>
  42. exit command: This is used to exit from the current shell. 
    exit
  43. clear command: This is used to clear the terminal screen. 
    clear
  44. ip Command: It is an updated version of the ipconfig command. It is used to assign an IP address, initialize an interface, disable an interface. 
    ip a or ip addr
  45. ping Command: It is an updated version of the ipconfig command. It is used to assign an IP address, initialize an interface, disable an interface. 
    ping <destination>
  46. Mail Command: The mail command is used to send emails from the command line. 
    mail -s "Subject" <recipient address>
  47. git init command: This is used to convert an existing, unversioned project to a Git repository or initialize a new, empty repository. 
    git init
  48. git add command: This git add command adds all the changes in the working directory to the staging area. It tells Git that you want to include updates to a particular file in the next commit 
    git add .
  49. commit command : This is used to save your changes to the local repository. 
    git commit -m "message"
  50. --version command: This is used to ckeck current version of any application. 
    <application> --version
  51. heroku login command: This is used to login in heroku deployment environment. 
    heroku login
  52. heroku create command: This is used to create remote repository in heroku. 
    heroku create
  53. git remote command: This is used to check current git remote repository. 
    git remote
  54. git push command: This is used to deploy and push git local repository to heroku server using remote repository. 
    git push heroku master
